# Excel 批量导入知识

知识工作台支持下载模板并批量导入 `.xlsx` 文件。

## 使用方式

1. 在“知识工作台”点击“下载导入模板”。
2. 在“知识导入”工作表填写数据。
3. 从“分类字典”工作表复制分类 ID，填写到“知识分类”列。
4. 点击“Excel 批量导入”并选择填写后的文件。
5. 查看成功数量和逐行失败原因；成功知识会进入待审核状态。

## 字段

| 字段 | 必填 | 说明 |
|---|---:|---|
| 标题 | 是 | 最多 256 个字符 |
| 知识分类 | 是 | 推荐填写分类 ID，也支持唯一分类名称或完整分类路径 |
| 正文 | 是 | 纯文本正文，最多 100000 个字符 |
| 副标题 | 否 | 多项使用中文分号 `；` 分隔 |
| 场景标签 | 否 | 多项使用中文分号 `；` 分隔 |
| 适用业务 | 否 | 多项使用中文分号 `；` 分隔 |
| 适用类目 | 否 | 多项使用中文分号 `；` 分隔 |
| 适用品牌 | 否 | 多项使用中文分号 `；` 分隔 |
| 适用机型 | 否 | 多项使用中文分号 `；` 分隔 |
| 机型个性化 | 否 | 支持“是/否”、`true/false`、`1/0` |

## 导入规则

- 单个文件最多 100 条、最大 5MB，仅支持 `.xlsx`。
- 每行独立校验和入库，单行失败不会回滚已经成功的其他行。
- 成功知识的来源标记为 `excel`，状态为待审核。
- 分类不存在、格式错误、疑似重复或完全重复的行不会入库，并返回对应错误原因。
- 导入过程复用人工新建知识的查重、向量生成和检索索引逻辑。
